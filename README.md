# 🌌 EssentialBoardOS

EssentialBoardOS는 웹 기술을 활용하여 구현된 차세대 대시보드 및 미니앱 생태계입니다. 사용자는 별도의 코딩 지식 없이도 노드 기반의 비주얼 프로그래밍 도구를 사용하여 자신만의 미니앱을 직접 제작하고 실행할 수 있습니다.

---

## 주요 기능

### 1. 시각적 노드 에디터 (Node-Based Logic)
- **무한 캔버스**: 줌/팬 기능을 지원하는 광활한 작업 영역에서 자유롭게 로직을 구성합니다.
- **다양한 노드 지원**: 논리, 제어, 데이터 처리뿐만 아니라 GPS, 배터리, 클립보드 등 하드웨어 API 연동 노드를 제공합니다.
- **함수 시스템**: 반복되는 로직을 함수(Function)로 정의하여 깔끔하게 관리할 수 있습니다.
- **레이어 시스템**: 복잡한 노드 다이어그램을 레이어로 나누어 관리할 수 있습니다.

### 2. 드래그 앤 드롭 UI 에디터
- 버튼, 텍스트, 입력창, 이미지 등 다양한 UI 컴포넌트를 제공합니다.
- **멀티 페이지**: 한 앱 내에서 여러 페이지를 구성하고 전환할 수 있습니다.
- **실시간 속성 편집**: 컴포넌트의 가시적인 속성(색상, 크기, 위치)을 즉각적으로 수정합니다.

### 3. 강력한 런타임 엔진
- 제작한 JSON 기반 미니앱을 실시간으로 해석하고 실행합니다.
- **플로팅 윈도우**: 앱을 창 모드로 실행하고 크기를 자유롭게 조절할 수 있습니다.
- **Gemini AI 연동**: Google Gemini API를 활용한 스마트한 기능을 미니앱에 추가할 수 있습니다.

### 4. 대시보드 생태계
- **퀵노트 (QuickNotes)**: 바탕화면에 메모를 포스트잇처럼 부착합니다.
- **개인화 설정**: 날짜/시간, 날씨 정보 위젯 및 사용자 정의 테마를 지원합니다.

---

## 기술 스택

- **Frontend**: React 19, Vite, Javascript (ESM)
- **Icons**: React-Icons
- **State Management**: React Hooks (useState, useEffect, useMemo, etc.)
- **Styling**: Vanilla CSS (CSS Variables 기반 테마 시스템)
- **Backend**: Node.js Proxy (AI 연동 및 외부 API 요청 처리)

---

## 시작하기

### 설치
```bash
npm install
```

### 실행 (클라이언트 + 서버 동시 실행)
```bash
npm start
```

- **클라이언트**: `http://localhost:5173`
- **서버**: `http://localhost:3000` (API 프록시 및 AI 기능 담당)

---

## 프로젝트 구조

- `src/framework`: 노드 에디터 및 런타임 엔진의 핵심 로직
- `src/components`: 대시보드 UI 및 기본 컴포넌트
- `src/apps`: 기본 내장된 계산기, 설정 등 표준 앱
- `server/`: 백엔드 프록시 및 API 서버
